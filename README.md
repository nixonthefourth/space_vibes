# Space Vibes

A passion project inspired by the cult-classic **Elite (1984)**.

An exploration of **NumPy** and **Pygame**, applying concepts from linear algebra and optics to render a 3D-ish space environment. Fly around in a spaceship modelled after the iconic **Sidewinder** from the original game — no Newtonian physics, just vibes.

## Project Structure

```
├── main.py
├── requirements.txt
├── .gitignore
└── README.md
```

## Getting Started

This project requires **Python 3.12.x** to ensure compatibility with NumPy and Pygame.

1. Clone the repository and navigate to the project folder
2. Create a virtual environment:
   ```bash
   python -m venv .venv
   ```
3. Activate it:
   - **Windows:** `.venv\Scripts\activate`
   - **macOS/Linux:** `source .venv/bin/activate`
4. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
5. Run the project:
   ```bash
   python main.py
   ```
