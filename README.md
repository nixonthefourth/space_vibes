# Space Vibes
Passion project referncing a cult-classic Elite 1984.

I have tried dabbling further into numpy and pygame with the application of linearly algebraical and optical proves and derivates.

The program allows you to fly around space in a spaceship mimicking an original Sidewinder.

Doesn't apply Newtonian Physics.

# Project Structure
```
| requirements.txt
| README.md
| main.py
```

# Running the Project
Make sure you use Python 3.12.x to make sure both numpy and pygame work. Create a .venv in your IDE and install all dependencies.
